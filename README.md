# 📊Australian Full-Time Students Overview (2006–2022) - Tableau 

## Project Overview
This data visualization project provides a comprehensive analysis of the Australian education landscape over a **16-year period**. Using **Tableau**, the dashboard tracks enrollment trends, demographic shifts, and regional distributions, offering actionable insights into the growth and composition of the Australian student body.

**View the Interactive Dashboard:** [Tableau Public Link](https://public.tableau.com/app/profile/phuong.anh.chu/viz/Dataviz1-dashb-v3/Dashboard1?publish=yes)  

---

## 🔍 Key Visualizations & Features

1. **Student Distribution Trend Area Chart**  
   - Visualizes the growth of **primary vs. secondary students** from 2006.  
   - **Interactivity:** "Select Filter" parameter allows toggling views by School Level.

2. **State-Level Demographics Multi-Dimensional Pie Chart**  
   - Breaks down student populations by Multiple variables and attributes: **Gender, Sector (Government vs. Non-Government), and School Level**.  
   - **Time Slider:** Year-by-year animation (2006–2023) shows evolving state distributions.

3. **Indigenous vs. Non-Indigenous Enrollment Dual-Axis Comparison**  
   - Analyzes enrollment patterns between **Indigenous and Non-Indigenous students**.  
   - **Technical Detail:** Independent Y-axis scaling handles population disparities (~10x) while preserving trend clarity.

4. **Gender Ratio Analysis Line Chart**  
   - Tracks **Male-to-Female ratio** over time across states (ACT, NSW, NT, QLD, etc.).  
   - **Contextual Logic:** Baseline set at 1.0 to clearly highlight state-specific fluctuations.

---

## 💡 Key Insights
- **Regional Dominance:** NSW consistently has the highest number of full-time students, followed by Victoria and Queensland.  
- **Stable Growth:** Student numbers show a steady increase across all sectors.  
- **Sector Preference:** Majority of students enrolled in the **Government sector** and **Primary education**.  
- **Demographic Ratios:** Non-Indigenous students outnumber Indigenous students by ~10:1, but both show similar upward trajectories.  
- **Gender Balance:** Student gender ratio remains highly balanced with only minor annual fluctuations.
